Go Live

Prepare a live session with scene planning, host notes, backup paths and stream readiness checks. This is the creator-facing control page for live sessions.

Pre-Live Checklist

  • Camera framing checked and audio return confirmed.
  • Host rundown loaded and guest call links distributed.
  • Backup recording path armed before stream launch.

Session Format

  • Opening bumper and lower-third naming sequence.
  • Main segment block with comments and audience prompts.
  • Close-out clip marker for post-show highlight editing.
